# Juan Gualotuña
## Instalacion 
- Instalar dependencias
```
npm install --legacy-peer-deps

```
Ya una vez instaladas las dependencias, ejecutamos el servidor que se alojara
```
ionic serve
```
## Apk:
El Apk se encuentra subido en el directorio.
